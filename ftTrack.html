Create user

URL: https://lobonode.ddns.net/ft/api/create

METHOD: GET

BODY: JSON {email: uniqueEmail, password:desiredPass}

Response: JSON {status:true, user} OR error

Login

URL: https://lobonode.ddns.net/ft/api/login

METHOD: POST

BODY: JSON {email: email, password:chosenPass}

Response: JSON {status:true, sessionDetails} OR if wrong credentials {status:false}

Check if user is logged in

URL: https://lobonode.ddns.net/ft/api/logged

METHOD: GET

BODY: NO BODY

Response: JSON {status:true, UserName} OR {status:false}

Logout user

URL: https://lobonode.ddns.net/ft/api/logout

METHOD: GET

BODY: NO BODY

Response: JSON {status:true, UserName} OR {message:error}

AddItem

URL: https://lobonode.ddns.net/ft/addItem

METHOD: POST

Body

{
ean: number UNIQUE REQUIRED
name: text REQUIRED
brand: text REQUIRED
weight: number
energy: number
carbs: number
sodium: number
fat: number
satfat: number
salt: number
sugar: number
contains: Array of text
organic: boolean
fairtrade: boolean
eco: boolean
vegetarian: boolean
vegan: boolean
healthnotes: text
}

Response: JSON {message: item added, status:true} OR ERROR

Get all items

URL: https://lobonode.ddns.net/ft/getItems

METHOD: GET

BODY: NO BODY

Response JSON of all items
